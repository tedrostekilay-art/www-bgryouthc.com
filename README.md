# Benishangul-Gumuz Region Youth Council Website

A modern, accessible website for the Benishangul-Gumuz Region Youth Council built with the **Unity & Progress** design system.

## 🎯 Overview

This website serves as a digital bridge between the regional government and young constituents, fostering civic empowerment, youthful energy, and institutional integrity.

### Design Pillars
- **Civic Empowerment** - Youth participation in regional governance
- **Youthful Energy** - Dynamic, innovative community development
- **Institutional Integrity** - Transparent and accountable operations

## 🎨 Design System Features

### Colors
- **Primary Blue (#1E73BE)** - Brand core, actions, headers
- **Secondary Orange (#F99D1C)** - High-priority CTAs and youth initiatives
- **Deep Navy (#0D2C4D)** - Typography and contrast
- **Neutral Greys** - Clean, institutional aesthetic

### Typography
- **Plus Jakarta Sans** - Modern headings
- **Inter** - Body text and UI elements
- **Noto Sans Ethiopic** - Amharic script support

### Layout
- **Desktop (1440px+)** - 12-column grid, 1200px max-width
- **Tablet (768-1439px)** - 8-column fluid, 32px margins
- **Mobile (≤767px)** - 4-column fluid, 20px margins

## 🚀 Getting Started

### Local Development

1. Clone the repository:
```bash
git clone https://github.com/tedrostekilay-art/www-bgryouthc.com.git
cd www-bgryouthc.com
```

2. Open `index.html` in your browser or use a local server:
```bash
python -m http.server 8000
# Visit http://localhost:8000
```

### Live Website

The website is deployed via GitHub Pages at:
**https://tedrostekilay-art.github.io/www-bgryouthc.com/**
